# Supply Chain Analytics Dashboard (Power BI)

## Project Overview
This project analyzes sales demand, inventory performance, and supplier delivery using Power BI.

## Key Metrics
- Total Sales
- Average Inventory
- Inventory Turnover
- Days of Inventory (DOH)
- OTIF %

## Dashboard Features
- Monthly sales demand trend
- Sales vs inventory comparison
- SKU level inventory analysis
- Inventory health distribution

## Tools Used
- Power BI
- DAX
- Excel
- Data Modeling
